Build a verified, comprehensive context pack from a folder.

Usage:
- `/deep-context /path/to/folder` — full build
- `/deep-context /path/to/folder --domain=legal` — with domain hint (legal|code|sales|research|finance|medical)
- `/deep-context ask /path/to/folder "your question"` — query cached pack
- `/deep-context link /folder-a /folder-b` — cross-link two packs

Invokes the deep-context skill. See SKILL.md for full documentation.
